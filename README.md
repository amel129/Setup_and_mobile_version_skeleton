### Prerequisites

In order to run this project you need:

- Computer
- Web Browswer
- Internet Connection

### Setup

Make sure you have Git and GitHub

Install this project with:

```sh
  cd my-folder
  git clone https://github.com/amel129/Setup_and_mobile_version_skeleton.git
```
### Usage

To run the project, open the html file on your repository using your web browser.
